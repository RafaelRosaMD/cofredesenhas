# Cofre de Senhas
Aplicativo mobile desenvolvido em React Native com Expo para cadastro e visualização segura de senhas, utilizando autenticação biométrica.

## Tecnologias utilizadas
- React Native
- Expo
- TypeScript
- Expo Local Authentication

## Funcionalidades
- Autenticação biométrica
- Cadastro de senhas
- Listagem de senhas
- Exibição/ocultação da senha
- Exclusão de registros
